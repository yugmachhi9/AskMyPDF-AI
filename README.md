# IntelliRAG 🚀

An AI-powered Retrieval-Augmented Generation (RAG) application that allows users to upload PDFs and ask questions using semantic search and LLM-based answering.

Built an end-to-end RAG-based AI application using LangChain, ChromaDB, HuggingFace embeddings, Mistral AI, and Streamlit for intelligent PDF question answering.
